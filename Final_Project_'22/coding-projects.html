Rock Paper Scissors

The goal of this exercise was to demonstrate the understanding of JavaScript and how we can use basic logic rules to create a simple rock paper scissor game. To start the human user chooses either a rock, paper or scissor option. Then the computer is assigned a random value. Depending on the value the computer was given at random, a winner is determined with and if, else if or else statement.
